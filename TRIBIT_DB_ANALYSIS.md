# Tribit 数据库分析报告

## 一、数据库概览

tribit.db 包含11个数据表，涵盖了内容洞察、KOL管理和自媒体运营等多个维度的数据。

### 表结构总览

| 表名 | 记录数 | 主要内容 |
|------|--------|----------|
| insight_search | 4,675 | 搜索关键词洞察数据 |
| insight_video_tk_creator | 76 | TikTok创作者分析数据 |
| insight_video_tk_product | 1,000 | TikTok产品销售数据 |
| kol_tribit_2024 | 107 | 2024年Tribit合作KOL |
| kol_tribit_india | 189 | 印度市场KOL数据 |
| kol_tribit_total | 189 | 全球KOL总表 |
| kol_ytb_video | 995 | YouTube频道详细数据 |
| selfkoc_ins | 362 | Instagram自媒体数据 |
| selfkoc_ytb | 332 | YouTube自媒体数据 |
| selkoc_account | 167 | 自媒体账号信息 |
| selkoc_tk | 475 | TikTok自媒体数据 |

## 二、表结构详细说明

### 1. 搜索洞察表 (insight_search)
**功能**：记录与Tribit品牌相关的搜索关键词数据
**主要字段**：
- `keyword`: 搜索关键词
- `search_volume`: 搜索量
- `cost_per_click`: 每次点击成本
- `modifier_type`: 修饰词类型
- `language`: 语言
- `region`: 地区

### 2. TikTok创作者表 (insight_video_tk_creator)
**功能**：分析TikTok上的创作者/达人数据
**主要字段**：
- `达人名称`: 创作者名称
- `达人粉丝数`: 粉丝数量
- `均播量`: 平均播放量
- `近30日销售额`: 最近30天的销售额
- `视频GPM`: 每千次播放收益
- `达人类型`: 创作者类型

### 3. TikTok产品表 (insight_video_tk_product)
**功能**：TikTok平台上的产品销售数据
**主要字段**：
- `商品名称`: 产品名称
- `商品价格最大值/最小值`: 价格区间
- `销量`: 销售数量
- `销售额`: 销售金额
- `带货达人数`: 推广该产品的达人数
- `商品类目`: 产品类别

### 4. KOL管理表 (kol_tribit_*)
**功能**：管理Tribit品牌合作的KOL信息
- **kol_tribit_2024**: 2024年合作的KOL列表
- **kol_tribit_india**: 印度市场的KOL
- **kol_tribit_total**: 全球KOL汇总

**主要字段**：
- `platform`: 平台（YouTube、Instagram、TikTok）
- `kol_account`: KOL账号
- `kol_post_url`: 发布内容链接
- `Region`: 地区

### 5. YouTube频道表 (kol_ytb_video)
**功能**：YouTube频道的详细分析数据
**主要字段**：
- `Youtuber`: 频道名
- `subscribers`: 订阅数
- `video views`: 视频观看数
- `category`: 频道类别
- `Country`: 国家
- `monthly_earnings`: 月收入估算
- `created_year`: 创建年份

### 6. 自媒体表 (selfkoc_*, selkoc_*)
**功能**：Tribit自有或密切合作的自媒体账号数据
- **selfkoc_ins**: Instagram账号数据
- **selfkoc_ytb**: YouTube账号数据
- **selkoc_tk**: TikTok账号数据
- **selkoc_account**: 账号汇总信息

**主要字段**：
- `selfkoc_account`: 账号名
- `Likes`: 点赞数
- `Comments`: 评论数
- `Views`: 观看数
- `selfkoc_post_date`: 发布日期

## 三、表关系分析

### 1. 平台维度关联
- 多个表通过`platform`字段关联（YouTube、Instagram、TikTok）
- 可以进行跨平台的KOL效果对比分析

### 2. 账号维度关联
- `kol_account`、`selfkoc_account`等字段可关联不同表
- 追踪同一账号在不同时期的表现

### 3. 时间维度关联
- 包含发布日期、创建日期等时间字段
- 支持时间序列分析

### 4. 地域维度关联
- Region、Country等字段支持地域分析
- 特别关注印度市场（独立表）

## 四、可进行的分析研究

### 1. 搜索洞察分析
- **关键词趋势分析**：识别高搜索量的产品相关关键词
- **用户意图分析**：通过modifier分析用户搜索意图
- **竞价策略优化**：基于CPC数据优化广告投放
- **多语言市场洞察**：分析不同语言市场的搜索特征

### 2. KOL营销效果分析
- **KOL画像分析**：粉丝量、播放量、互动率等多维度评估
- **ROI分析**：销售额与粉丝量、播放量的相关性
- **平台对比**：YouTube vs Instagram vs TikTok效果对比
- **地域化策略**：特别是印度市场的KOL策略

### 3. 产品销售分析
- **爆款产品识别**：基于销量和销售额数据
- **价格策略分析**：价格区间与销量的关系
- **品类分析**：不同产品类目的表现
- **达人带货效果**：分析带货达人数与销售的关系

### 4. 内容营销分析
- **内容表现追踪**：点赞、评论、观看数的趋势
- **发布时间优化**：分析最佳发布时间
- **内容类型效果**：不同平台内容类型的表现差异
- **互动率分析**：评论/观看比例等指标

### 5. 竞争情报分析
- **行业标杆研究**：通过YouTube大V数据了解行业标准
- **品类趋势洞察**：音响类产品在各平台的表现
- **营销策略借鉴**：成功案例的分析和学习

### 6. 市场扩展分析
- **地域机会识别**：基于不同国家/地区的数据
- **平台优先级**：确定不同市场的主要平台
- **本地化策略**：针对特定市场（如印度）的策略制定

### 7. 预测分析
- **销售预测**：基于历史数据预测未来销售
- **KOL成长预测**：识别潜力KOL
- **趋势预警**：及时发现市场变化

### 8. 综合报表
- **营销仪表板**：整合多维度数据的实时监控
- **月度/季度报告**：定期业务回顾
- **专题分析报告**：针对特定问题的深度分析

## 五、数据质量说明

### 优势
1. 数据覆盖面广：涵盖主流社交媒体平台
2. 多维度数据：从搜索到销售的完整链路
3. 时效性较好：包含近期数据

### 需要注意
1. 部分表存在未命名字段（Unnamed列）
2. 数据量差异较大（76条到4675条）
3. 部分字段数据类型可能需要清洗（如Views字段）

## 六、建议的分析优先级

1. **高优先级**
   - TikTok产品销售分析（直接关联收入）
   - 搜索关键词优化（获客成本优化）
   - KOL效果评估（营销投入优化）

2. **中优先级**
   - 内容表现分析（提升互动率）
   - 地域市场研究（扩展策略）
   - 竞品情报分析（市场定位）

3. **长期价值**
   - 预测模型构建
   - 自动化报表系统
   - 数据质量持续优化
